### 3. 무료/유료 버전, 확장 방향
- 무료버전: 위에 담긴 기능 구현
- 유료버전: 정해진 시간에 글 작성해서 저장하지 않으면 글 작성해서 저장할 때까지 알람이 반복적으로 울리게 하기(철저한 관리 기능) 
- 확장방향: 걷기(gps 활용) 등 사람들이 습관을 만들고 싶어하는 활동들을 앱에서 관리할 수 있도록 확장하는 방안 생각해 보기
