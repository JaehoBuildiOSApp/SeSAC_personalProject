### 3. 기타
- 앱을 처음부터 끝까지 만드는 경험이 굉장히 소중할 것 같다.
- 팀원들이 계획하고 있는 앱들보다 단순할 것으로 예상됨..
- 앱을 만들면서 제대로 복습하지 못했던 걸 다시 살펴보는 기회로 삼기
- 다른 사람들 작업물 보면서 누군가가 필요로 할 앱을 구상하기
- 시간 관리 능력 키우기(가정 == 개인프로젝트 >>> 직장 순으로 시간, 힘 배분)
